DETAIL

header영역은 addClass를 활용하여 상단에서 스크롤 탑값이 10px이상이면 header높이가 바뀌고 창 너비 990px일 때 텍스트 메뉴버튼 대신 3선 메뉴버튼이 나타납니다.

1번째 섹션은 4초마다 자동으로 넘어가는 슬라이드로, 너비와 높이는 창 크기에 맟춰 창을 줄여도 그 크기에 맞게 반응하도록 제작되었습니다.

2번째 섹션은 창 너비 767px일 때 float를 해제하여 오른쪽 이미지박스를 왼쪽 텍스트박스 아래로 배치하였습니다.

3번째 섹션은 창 너비 1199px일 때 텍스트박스의 아이콘과 텍스트 묶음이 1개씩 배치하였으며, 990px일 때는 float를 해제하여 오른쪽 이미지박스를 왼쪽 텍스트박스 아래로 배치함과 동시에 아이콘과 텍스트 묶음을 다시 2개씩 배치하였습니다.
창 너비가 767px일 때는 텍스트박스의 아이콘과 텍스트 묶음이 1개씩 배치되도록 제작되었습니다.

4번째 섹션은 한 행의 이미지 개수가 창 너비 990px초과일 때는 4개, 990px이하에서는 2개, 767px일 때는 1개로 제작되었습니다.

6번째 섹션은 한 행의 이미지 개수가 창 너비 767px초과일 때는 3개, 767px이하에서는 1개로 제작되었으며, 이미지 클릭 시 해당 이미지가 슬라이드 형식의 모달 창으로 나옵니다.

7번째 섹션은 해당하는 숫자만큼 카운트 되는 효과를 넣었습니다.

9번째 섹션은 4개의 이미지 슬라이드가 창 너비 990px에서는 3개의 이미지 슬라이드와 페이지 버튼이 추가되며, 767px에서는 1개의 이미지 슬라이드로 제작되었습니다.

11번째 섹션은 이미지에 hover 시 흑백효과를 주었으며, 3개의 이미지가 창 너비 767px일 때 float를 해제하여 1개씩 보여주도록 제작되었습니다.

12번째 섹션은 정규표현식을 활용하여 이름,메일주소,전화번호 등을 기입해 폼전송을 할 수 있는 섹션으로 구성되었습니다.
