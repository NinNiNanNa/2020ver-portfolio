DETAIL

header, footer영역은 창 너비가 1120px에 도달하면 모바일 버전의 header, footer로 바뀝니다.

1번째 섹션은 동영상으로 시작되며 영상 하단에 소리를 on/off 기능과, 재생/일시정지 버튼, 아래 화살표를 클릭 시 다음 섹션으로 이동하는 스무스 스크롤링 버튼 이벤트로 구성되었습니다.

2번째 섹션은 텍스트 줄의 수로 인해 좌측 박스와 우측 박스의 높이가 달라져 창 너비가 1114px 초과일 때는 좌측 박스 높이에 맟추게 제작되었습니다.

3번째 섹션은 동영상들이 나열되어 있으며, 각 각의 동영상에 mouseenter 시 설명이 나옴과 동시에 동영상이 정지되고 mouseleave 시 동영상이 재시작됩니다.

섹션 상단에 네비게이션 바를 배치하여 해당 메뉴를 누르면 메뉴에 해당하는 동영상들이 나오도록 제작되었으며, LOAD MORE 버튼을 클릭 시 숨겨져 있던 동영상들이 나옵니다.
